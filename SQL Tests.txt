1. Write a query to list all of the orders placed by customers with the surname “Walker”
////////////////////////
SELECT customers.forename, customers.surname, orders.*
FROM orders
INNER JOIN customers on orders.customer_number=customers.customer_number
WHERE customers.surname = 'Walker'
////////////////////////
296 Results.

2. The total value of all invoices settled in April 2020
/////////////////////////////////////////////////
SELECT sum(invoice_value) FROM invoices WHERE settlement_date LIKE '2020-04%'
///////////////////// or ////////////////////////
SELECT sum(invoice_value) FROM invoices WHERE settlement_date BETWEEN '2020-04-01' AND '2020-04-30'
/////////////////////////////////////////////////
Total Value = 1819553.62 (In both cases)
